🛠️ Pokémon GO Admin
레이드 입력 어드민
여기서 입력한 정보를 저장하면 메인, 지금 레이드, 주간 일정표, 월간 일정, 상세 일정 페이지에서 공통으로 불러와서 표시할 수 있게 만드는 기준 페이지입니다. 현재 버전은 localStorage 저장 방식이라서 바로 테스트하기 좋습니다.
저장키 : pogoRaidAdminData 기본 방식 : 브라우저 저장 나중에 JSON 방식으로 확장 가능
입력 폼
보스/이벤트 하나를 입력하거나 수정하는 영역입니다. 저장 후 오른쪽 목록과 미리보기에서 바로 확인됩니다.
새 일정 입력

비워두면 기본값 10:00 PM
공략 타입 입력
예: 얼음 / 드래곤 / 페어리, 그리고 원형 타입 아이콘 URL
저장된 일정 목록
수정 / 삭제 / 복제 / 정렬을 여기서 할 수 있습니다.

const saved = localStorage.getItem('pogoRaidAdminData');
const raidData = saved ? JSON.parse(saved) : [];
