GUIDE
이 페이지는 브라우저에서 임시 편집 후 raid-data.json 파일로 내보내는 용도입니다.
최종 반영은 내보낸 JSON을 사이트 루트에 업로드하면 됩니다.
불러오기 예시
const res = await fetch('./raid-data.json');
const raidData = await res.json();
