데이터 불러오기 / 저장
현재 raid-data.json 불러오기 또는 JSON 붙여넣기로 시작할 수 있습니다.
저장은 브라우저가 같은 폴더 파일을 직접 덮어쓸 수 없어서, 다운로드한 파일을 기존 raid-data.json으로 다시 업로드해 교체하는 방식입니다.
